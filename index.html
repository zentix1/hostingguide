Free Hosting Guide with Vercel

Prepare your project Purchased Tool Source Code folder .
Go to GitHub and log in (or create an account).
Click "New Repository", give it a name (like my-site), and click Create repository.
Upload your project files to this GitHub repo. You can drag & drop files or use GitHub Desktop for easier management.
Now, go to Vercel and sign in using your GitHub account.
Once logged in, click on "Add New → Project". It will show your GitHub repositories.
Select the repository you just uploaded, and Vercel will auto-detect it. You can review the settings or leave them default.
Click the "Deploy" button. Wait a few seconds while Vercel builds your site.
Once done, Vercel will give you a live link like: https://your-project-name.vercel.app. You can share this link with anyone — your project is now live online!
To update your site later, just push changes to your GitHub repo — Vercel auto-updates your site.
